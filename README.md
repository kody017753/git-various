# github

**「GitHub」** とは、バージョン管理システムの **「Git」** を利用して、ソフトウェア開発プロジェクトのソースコード管理を共有することができる **Web サービス** です。  
今回、参考資料を一括で確認するために登録が必要！

## 1.「GitHub」のアカウントを作成する方法

### 1-1 GitHub にアクセスしてアカウント情報を登録する

- 「GitHub」の公式サイト (https://github.com/) にアクセス
